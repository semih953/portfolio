Gestion des Équipements de Protection Individuelle (EPI)

Description du projet

Ce projet a consisté à développer une application complète pour la gestion des Équipements de Protection Individuelle (EPI), permettant aux entreprises de suivre les attributions, les maintenances et la conformité aux normes de sécurité.

Objectifs

L'objectif principal était de créer une solution efficace pour la gestion des équipements de protection individuelle permettant d'assurer le suivi des attributions et le respect des normes de sécurité.

Solutions proposées

J'ai créé une application Windows avec C# et WPF, connectée à une base de données SQL Server pour stocker et gérer les données des équipements et des utilisateurs. L'interface a été conçue pour être intuitive et adaptée aux besoins des responsables sécurité.

Résultats obtenus

Cette application a permis d'améliorer significativement le suivi des EPI, facilitant le travail des responsables sécurité et garantissant une meilleure conformité aux normes en vigueur. Les délais d'intervention pour la maintenance des équipements ont été réduits et la traçabilité des attributions a été grandement améliorée.

Technologies utilisées

  • Langages : C#, SQL
  • Framework : WPF (Windows Presentation Foundation)
  • Base de données : SQL Server
  • ORM : Entity Framework
  • Outils de développement : Visual Studio, SQL Server Management Studio
