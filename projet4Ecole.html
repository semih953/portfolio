Liste de Courses

Un gestionnaire de liste de courses simple en Python

À propos du projet

Ce projet est une application console Python permettant de gérer une liste de courses. Vous pouvez facilement ajouter, supprimer ou modifier des articles et leurs quantités. Une solution idéale pour organiser vos achats quotidiens !

Fonctionnalités

Ajouter

Ajoutez de nouveaux articles à votre liste avec leur quantité.

Supprimer

Supprimez facilement des articles de votre liste de courses.

Modifier

Modifiez les articles et les quantités déjà présents dans la liste.

Comment utiliser

  1. Clonez le dépôt ou téléchargez le fichier liste_courses.py
  2. Exécutez-le dans un terminal avec la commande py liste_courses.py
  3. Suivez les instructions dans la console pour gérer votre liste de courses

Le programme offre une interface simple à utiliser avec des options numérotées. Sélectionnez l'action que vous souhaitez effectuer et suivez les instructions !

Améliorations possibles

  • Persistance des données (sauvegarde dans un fichier)
  • Interface utilisateur graphique
  • Catégorisation des articles
  • Calcul du prix total estimé
