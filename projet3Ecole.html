Jeu d'Aventure Textuelle

Une expérience interactive en Python

Description du projet

Un jeu d'aventure textuelle se déroulant dans un lycée spécial nommé A.U. où les élèves possèdent des super-pouvoirs. Le joueur doit explorer différents lieux, interagir avec l'environnement et accomplir des missions pour progresser.

Technologies utilisées

Python POO Structures de données Interface console

Fonctionnalités

  • Création de personnage avec choix de pouvoir
  • Navigation entre différents lieux de l'académie
  • Système de combat contre des mannequins d'entraînement
  • Gestion d'inventaire et des points de vie
  • Collection de badges pour progresser dans le jeu
  • Système d'alimentation pour récupérer des PV
