Les4Fantastiques

Explorez l'univers Marvel comme jamais auparavant

À propos du projet

Les4Fantastiques est une application web qui permet aux fans de Marvel de découvrir l'univers riche des super-héros, des méchants et des séries. Notre plateforme offre une expérience immersive pour tous les passionnés de l'univers Marvel.

Personnages

Explorez les profils détaillés des héros et vilains Marvel, avec leurs histoires et leurs pouvoirs.

Séries

Découvrez les séries emblématiques de l'univers Marvel et leur chronologie.

Mini-jeu

Testez vos réflexes avec notre mini-jeu inspiré de l'univers Spider-Man.

Technologies utilisées

Notre projet est construit avec des technologies modernes pour offrir une expérience utilisateur fluide et réactive :

Python Django HTML5 CSS3 JavaScript Bootstrap API Marvel MySQL

Installation

Pour installer et exécuter le projet localement :

# Cloner le dépôt
git clone https://github.com/Semiiih/Les4Fantastiques.git
cd Les4Fantastiques

# Configurer la base de données
# Importer le fichier les4fantastiques.sql

# Lancer le serveur
python3 manage.py runserver

Fonctionnalités principales

Gestion des comptes

Créez votre compte pour sauvegarder vos personnages et séries préférés.

Liste de favoris

Ajoutez vos personnages et séries préférés à votre liste personnelle de favoris.

Mini-jeu interactif

Un jeu amusant où vous contrôlez Spider-Man sautant au-dessus des obstacles.

Quiz Marvel

Testez vos connaissances sur l'univers Marvel avec notre quiz interactif.
